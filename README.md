# App Creator

Generates a node app.

## Usage

`npx github:iortega/create-app`

## Basic Setup and Options

By default the generator setup:

- Babel
- CSS: PostCSS, Sass
- Prettier
- ESLint

You can choose from different technologies:

- **Package Manager:** npm, yarn
- **Bundler:** webpacker

And choose if you want:

- Typescript?
- React?
- ExpressJS?
